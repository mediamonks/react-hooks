# useUpdate

Hook that returns a callback that forces an update, the callback can be awaited.

## Reference

```ts
function useUpdate(): () => void;
```

### Returns

A callback that forces an update, the callback can be awaited.

## Usage

```tsx
function DemoComponent() {
  const update = useUpdate();

  useEffect(() => {
    (() => {
      await update();
      console.log('After update');
    })();
  }, [update]);

  return <div></div>;
}
```
